Types of Cloud Computing There are 3 main types of Cloud Computing, IaaS-Infrastructure-as-a-Service, PaaS-Platform-as-a-Service, and SaaS-Software-as-a-Service. There are also many other small, less-used versions of the Cloud.

IaaS

IaaS, which stands for Infrastructure as a Service, is a LOCATION type of cloud.

  • Provisioned and managed by the Internet
  • On-premises handling
